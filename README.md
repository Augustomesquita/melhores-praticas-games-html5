# Melhores Práticas de JavaScript para Jogos em HTML5
Esta apresentação apresenta algumas diretrizes e recomendações para otimizar o uso de JavaScript no desenvolvimento de jogos baseados em HTML5. Muitas destas recomendações são aplicáveis a qualquer aplicação web ou site que utilize JavaScript, enquanto a segunda parte é focada especificamente para o uso de JavaScript e manipulação do Canvas.
